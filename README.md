<h2>Restaurant Matching</h2><br>
<h3>1.サービス概要</h3><br>
訪日外国人と日本人のマッチングサービスです。日本人が訪日外国人に向けてオススメの飲食店の情報を投稿してマッチングしたらガイドとして案内してあげる仕組みです。<br>
<h3>アプリケーション機能一覧</h3><br>
  お気に入り登録機能（Ajaxによる非同期通信）,
  記事一覧表示,記事詳細表示,記事投稿,
  管理ユーザー登録,管理ユーザーログイン,Facebookアカウントログイン機能,
  DM機能,
  フォロー機能,
  画像ファイルアップロード機能,
  ページネーション機能,
  検索機能,
  Googlemap表示機能,
